# Backend

ExpressJS + firebase

### Installation:

Using pnpm here, highly recommending installing it as it is much faster.

```
pnpm i
```

Copy `sample.env` to `.env` and fill out the details. [Docs](https://support.google.com/firebase/answer/7015592#web&zippy=%2Cin-this-article)

### Running:

```
pnpm run dev
```
